Equation Calculator

My First Major Project

When I was 18 and first beginning my journey into programming, I was particularly ambitious with projects that I wanted to create. One of which was a calculator that not only did the normal functionality but set up all of the formulas for you with minimal input from the end-user. This project was 9 years ago back when phone apps were starting to become more and more powerful and although I knew that other products were on the market, this was my own and tested my own ability to make a phone app. The project started out rather small getting the main functions to work, some basic formulas, and, ultimately, learning how to make an Android application. As I progressed in my courses in college, I learned more/better techniques and added what I could to the application and transformed it into a massive calculator that performed hundreds of equations. It was not always the prettiest application to use, but it was more about the learning experience of making it that I adhered to. A LOT of the code in the application was incredibly poor, unoptimized, and all-around ugly to look at but, again, this was the first real coding experience I had and was evolving with me. In the end, the calculator was able to do simple calculus, geometry, algebraic equations, unit conversions, trigonometry, physics equations, and chemistry equations. The main premise of these equations was that the user enters in the information that they knew and the calculator would automatically solve for what was missing or let them know that it needs more information.
